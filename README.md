# 🍹 Cocktail Finder App

A full-stack Node.js application that uses the CocktailDB API to search and display cocktail recipes.

---

## 🚀 Features
- Search cocktails by name
- Fetch data from external API (CocktailDB)
- Display drink image, name, and instructions
- Responsive card layout
- Error handling for invalid searches
- Clean UI with CSS styling

---

## 🛠 Tech Stack
- Node.js
- Express.js
- EJS (Templating Engine)
- Axios (HTTP requests)
- HTML / CSS

---

## 📡 API Used
https://www.thecocktaildb.com/api.php
